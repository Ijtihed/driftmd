# Changelog

## 0.1.0

Initial release.

- File reference checking
- Internal link checking
- Directory tree validation
- CLI flag verification (commander, argparse, click, clap)
- Environment variable cross-referencing (JS, Python, Rust, Ruby, Go, Java)
- Badge version checking
- Function signature validation (opt-in)
- GitHub Action with PR comments and SARIF output
- Zero-config defaults with `.driftmdrc.json` overrides
- `--severity` filtering and `--json` output for CI
